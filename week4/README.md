# Activities Completed

## 1. Threat Hunting Practice

* Developed hunting hypothesis for unauthorized privilege escalation
* Queried Event ID 4672 in Elastic Security
* Investigated suspicious indicators using AlienVault OTX
* Validated findings using Velociraptor process queries
* Mapped findings to MITRE ATT&CK T1078

## 2. SOAR Playbook Development

* Created automated phishing response playbook
* Integrated CrowdSec IP blocking
* Generated TheHive incident tickets
* Tested automated response workflow

## 3. Post-Incident Analysis

* Conducted Root Cause Analysis using 5 Whys
* Created Fishbone Diagram in Draw.io
* Calculated MTTD and MTTR metrics
* Documented lessons learned

## 4. Alert Triage with Automation

* Investigated suspicious file download alerts in Wazuh
* Automated VirusTotal hash verification
* Managed incidents through TheHive
* Prioritized alerts based on severity

## 5. Evidence Analysis

* Collected and analyzed network connection evidence
* Performed Velociraptor netstat analysis
* Maintained chain-of-custody records
* Generated SHA256 evidence hashes

## 6. Adversary Emulation Practice

* Simulated spearphishing attack using MITRE Caldera
* Configured Wazuh detection rules
* Verified alert generation and response
* Identified detection gaps

## 7. Security Metrics and Executive Reporting

* Created Elastic Security dashboards
* Calculated MTTD, MTTR, and dwell time
* Prepared executive SOC performance reports
* Documented improvement recommendations

## 8. Capstone Project

* Simulated Samba exploit attack using Metasploit
* Detected malicious activity with Wazuh
* Automated containment using CrowdSec
* Performed incident triage in TheHive
* Conducted RCA and post-incident reporting
* Generated executive stakeholder briefing
