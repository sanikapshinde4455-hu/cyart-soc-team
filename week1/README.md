## Overview

This repository contains theoretical and practical SOC (Security Operations Center) learning activities completed as part of SOC Analyst training. The project covers security monitoring, log analysis, incident response, threat hunting, alert triage, adversary emulation, evidence analysis, and security reporting.

## Objectives

* Understand SOC operations and workflows.
* Learn security monitoring and log management.
* Practice incident detection and response.
* Analyze security events and alerts.
* Configure monitoring dashboards and alert rules.
* Document incidents and security findings.

## Practical Activities

### Log Analysis

* Investigated Windows Event IDs 4625 and 7045.
* Identified failed login attempts and suspicious activities.

### Alert Triage

* Analyzed security alerts in Wazuh.
* Validated indicators using VirusTotal.
* Documented findings in TheHive.

### Threat Hunting

* Developed hunting hypotheses.
* Queried logs using Elastic Security.
* Investigated privilege escalation events.

### Incident Response

* Simulated attacks using Metasploit.
* Performed containment and remediation actions.
* Conducted post-incident analysis.

### Adversary Emulation

* Simulated MITRE ATT&CK techniques using Caldera.
* Tested detection capabilities in Wazuh.

### Evidence Analysis

* Collected and analyzed forensic evidence using Velociraptor and FTK Imager.
* Maintained chain-of-custody documentation.

### Security Metrics & Reporting

* Calculated MTTD and MTTR.
* Created dashboards and executive reports.
